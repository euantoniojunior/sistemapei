# Sistema PEI - SENAC

Sistema para criação, edição e acompanhamento de Planos Educacionais Individualizados (PEI) de alunos com necessidades especiais.

## Tecnologias
- Backend: Python + Flask
- Frontend: HTML + CSS + JS
- Banco de Dados: SQLite (alterável para PostgreSQL)

## Como rodar

1. Instale as dependências:
pip install -r requirements.txt

2. Inicie o servidor Flask:
python backend/app.py

3. Acesse: http://localhost:5000/
